📊 Accenture Data Analytics Virtual Internship – Portfolio
🔎 Overview
This virtual internship with Accenture (Forage) provided hands‑on experience in data cleaning, modeling, visualization, and storytelling. The project focused on analyzing Social Buzz, a fast‑growing social media platform, to uncover insights into user engagement and content popularity.

📝 Tasks Completed
Task 1: Project Understanding
Reviewed the Social Buzz business problem and defined requirements.

Identified key deliverables: audit of big data practice, IPO recommendations, and content popularity analysis.

Drafted executive‑level questions for the CEO and CMO to guide analysis.

Task 2: Data Cleaning & Modeling
Merged multi‑source datasets (Users, Content, Reactions, Reaction Types) into a unified data model.

Applied formulas (VLOOKUP, SUMIF) to join tables and calculate aggregate popularity scores.

Produced a clean dataset with sentiment and engagement metrics ready for analysis.

Task 3: Data Visualization & Storytelling
Built interactive dashboards in Power BI and Excel to visualize:

Top 5 content categories (Technology, Science, Food, Animals, Healthy Eating).

Sentiment distribution across reactions.

Monthly posting trends and engagement spikes.

Designed a consulting‑style presentation using Accenture’s template to communicate insights clearly.

Task 4: Communicating Insights & Analysis
Delivered a structured presentation to executives (CEO & CMO).

Highlighted actionable insights:

Science & Technology drive the highest engagement.

Food and Healthy Eating offer collaboration opportunities with brands.

February showed peak posting activity, suggesting seasonal engagement trends.

Recommended scaling the analysis into real‑time production dashboards for continuous monitoring.

🎯 Key Insights
Science & Technology are the most popular categories, reflecting user interest in factual content.

Food & Healthy Eating are common themes, offering opportunities for brand partnerships.

Seasonal posting trends indicate timing strategies can boost engagement.

🧠 Skills Applied
Data Cleaning & Modeling: SQL, Excel (VLOOKUP, SUMIF)

Data Visualization: Power BI, Tableau, Excel

Analytics: KPI frameworks, sentiment analysis, cohort analysis

Business Storytelling: Executive presentations, consulting‑style insights
